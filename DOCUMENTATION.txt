# Liss.py

## Installation

Pour installer, tout d'abord installez une distribution Python, via par exemple Anaconda:

https://docs.conda.io/en/latest/miniconda.html

Ouvrez un terminal dans le dossier de travail.

Pour installer les dépendances, entrez:

$ make install

Pour le fichier d'entrée ainsi que les paramètres des Cp, ceci est indiquer dans le fichier configuration.ini

Ce fichier est à modifier à votre convenance.

Ensuite, entrez

$ make run 

dans un terminal pour faire tourner le programme.
